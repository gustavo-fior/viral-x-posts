# viral-x-posts
A Claude skill for founders and builders who want to launch products on X and actually get seen. Drop it into any Claude setup that supports skills and Claude will start drafting X posts and launch video scripts that follow the patterns that actually work on tech Twitter — not generic "we're thrilled to announce" corporate copy.
What's inside:

Account warm-up tactics — community tagging, daily posting cadence, copying trending templates from competitors, and honest timelines (spoiler: the breakout usually takes months, not days).
The 60-second launch video formula — a time-coded structure covering the one-second visual hook, the 3–7 word product line, the 10-second problem-to-demo rule, and why you should ship one feature per video instead of a full tour.
Founder-voice post heuristics — hook-first writing, screenshots over links, engagement-gated CTAs, and a filler-word hit list.
When not to use X — if the ICP isn't technical, the skill tells Claude to redirect the user to LinkedIn instead of forcing it.

The skill is a single SKILL.md file — no scripts, no dependencies. Install it wherever your Claude reads skills from and it triggers automatically on requests like "help me write a launch tweet," "script my demo video," or "how do I go viral on X."
